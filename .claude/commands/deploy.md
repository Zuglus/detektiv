Опубликовать сайт детективного агентства на сервер.

## Процесс

1. **Сборка** — запусти `hugo` в директории `/Users/z/detektiv`
   - Должно собраться 85 страниц (45 RU + 40 EN), 0 ошибок
   - Если сборка упала — остановись и сообщи об ошибке, не продолжай деплой

2. **Деплой** — запусти `npm run deploy` в директории `/Users/z/detektiv`
   - Скрипт читает `.deployconfig` (SFTP-данные, не коммитить)
   - Загружает папку `public/` на сервер через lftp/SFTP
   - Ожидаемый результат: `✅ Успешно загружено на сервер!`

3. **Сообщи результат** — URL сайта и статус деплоя

## Важно

- `.deployconfig` содержит пароли — никогда не показывай его содержимое и не коммить
- Если `lftp` не установлен — скрипт сам установит через brew
- Деплой заблокирован из экспериментальных веток — только из `main`
- Таймаут на загрузку может быть большим (много файлов) — подожди завершения
